# Electron React Starter with Hot Module Replacment

## Run in dev:

First start webpack dev server

```
 npm run dev:server
```

PS. Don't bother opening url in browser it will crash.

Next start Electron

```
npm start
```

## Build for production:

Build React app:

```
npm run prod
```

Then build Electron app:

```
npm run build-all
```
